The file RTLSRC.EXE contains the latest source code to the Symantec C++ 6.00
runtime libraries. This file has been uploaded by Symantec Corporation for 
registered users of their Symantec C++ 6.00 product.

To install this patch:

1) download the file RTLSRC.EXE
2) run RTLSRC.EXE in your SRC\CLIB directory (answer Y to any prompts)
3) run the batch file UPDATE.BAT
4) If successful, .RTPatch will not report any errors
5) OPTIONAL: delete the files PATCH.EXE, UPDATE.BAT, RTLSRC.RTP, and
   RTLSRC.EXE
