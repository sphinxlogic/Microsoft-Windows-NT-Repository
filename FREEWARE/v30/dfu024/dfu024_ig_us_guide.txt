
	This directory contains the kit for DFU version V2.4.
	The DFU_FREEWARE_024.A file is the kit which must be installed
	with the VMSINSTAL procedure. The complete installation
	procedure is described in the documentation. DFU
	can be installed on Alpha and VAX platforms.

	The [.DOC] directory contains the DFU documentation in
	Lineprinter, PostScript(TM) , Bookreader and HTML format.
	These documentation is also in the DFU_FREEWARE_024.A saveset, 
        and can optionally be installed to the SYS$HELP directory
	during installation.

	The [.LIS] directory contains the DFU sources listings.
