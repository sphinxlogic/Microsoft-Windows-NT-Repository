XCOLOREDIT 1.1 under OpenVMS
============================

You can run Xcoloredit via FREEWARE_DEMO.COM procedure.

Sources are located into [.SRC] directory. AXPOBJS.ZIP and VAXOBJS.ZIP archives
contain object files for relinking if you don't haev a C compiler.
