wxBuilder 1.9 for Windows
-------------------------

Installation
------------

Unzip the archive and install an icon for wxBuilder
in Program Manager.

If you do not have ctl3dv2.dll installed in your windows\system
directory, copy it there. Then delete it from the wxBuilder
directory on your hard disk, or wxBuilder may not run.

Please see the Change Log section of the help file for recent changes.

Good luck!
