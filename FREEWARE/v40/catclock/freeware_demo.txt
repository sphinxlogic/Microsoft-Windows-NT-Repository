Catclock port under OpenVMS                 May 1995
===========================

You need to place the resource file XCLOCK.DAT either into your SYS$LOGIN
directory or (better) into SYS$COMMON:[DECS$DEFAULTS.USER] directory (with
World:RE protection). You can customize this resource file.

You can run catclock via FREEWARE_DEMO.COM procedure.

Sources are available under [.SRC] directory in CATCLOCK.ZIP file
