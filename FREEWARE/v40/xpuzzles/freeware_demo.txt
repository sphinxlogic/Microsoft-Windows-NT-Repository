X Window Puzzles under OpenVMS
==============================

You can run 10 demos via FREEWARE_DEMO.COM procedure:

XCUBES
XDINO
XHEXAGONS
XMBALL
XMLINK
XOCT
XPYRAMINX
XRUBIK
XSKEWB
XTRIANGLES

Sources are into [.SRC] directory, where each puzzle program has its own
subdirectory. AXPOBJS.ZIP and VAXOBJS.ZIP archives contain object files for
relinking if you don't have a C compiler.
