TMESIS SYMBOL, UTILITIES, Set/Delete/Display DCL symbols+++

TMESIS SYMBOL V4.1%8.295 is a utility which allows the user to set, delete
and display DCL symbols in any process in a VMScluster (providing the user
has the appropriate privileges and rights to do so).  In addition,  SYMBOL
has many unique features which provide sub-procedural symbol manipulation;
features not available in native DCL!

SYMBOL V4.1%8.295 is provided as a VMSINSTALlable kit.  For additional in-
formation on the features and use of TMESIS SYMBOL V4.1%8.295, consult the 
$$$SYMBOL.INFORMATION document in the SYMBOL041.A saveset.
