VAX X11R4 DECUS Libraries:  in [.SRC]X11R4_DECUS_LIB_VAX.ZIP file
=========================
This kit contains ONLY the X11R4 libraries (XAW, XMU and XVMSUTILS) with
sources, object libraries (VAX C 3.2 and DEC C 4.0) and sharable images,
for VAX VMS. Client, contrib, demos, games etc can be found on DECUS tapes
(try http://www.decus.org/)

AXP X11R4 DECUS Libraries: in [.SRC]X11R4_DECUS_LIB_AXP.ZIP file.
=========================
This kit contains ONLY the X11R4 libraries (XAW, XMU and XVMSUTILS) with
sources, object libraries (DEC C 4.0), for Alpha VMS. Client, contrib, demos, 
games etc can be found on DECUS tapes (try http://www.decus.org/)
