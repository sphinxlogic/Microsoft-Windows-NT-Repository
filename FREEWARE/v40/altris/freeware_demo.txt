ALTRIS Tetris under OpenVMS
===========================

You can run 4 demos via FREEWARE_DEMO.COM procedure:

ALTETRIS 
HEXTRIS
TERTRIS
WELLTRIS

Sources are available into [.SRC] directory (each tetris has its own
subdirectory). AXPOBJS.ZIP and VAXOBJS.ZIP files contain object files for
relinking if you dont' have a C compiler.
