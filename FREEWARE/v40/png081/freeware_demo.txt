All sources and object libraries of png and zlib libraries are available in
[.SRC] directory into PNG081.ZIP file. You can extract them with UNZIP utility.

Files needed for compiling and linking:

PNG.H
LIBPNG.OLB            (GCC 2.5.8 on VAX/VMS 5.5-2)
LIBPNG.OLB_DECC       (DEC C 5.0 on VAX/VMS 6.1)
LIBPNG.OLB_AXP        (DEC C 5.0 on Alpha/VMS 6.1)

==> you may need GCCLIB.OLB for linking when using the GCC version of PNG 
on VAX.

ZCONF.H
ZLIB.H
LIBGZ.OLB             (VAX C 3.2 on VAX/VMS 5.5-2)
LIBGZ.OLB_DECC        (DEC C 5.0 on VAX/VMS 6.1)
LIBGZ.OLB_AXP         (DEC C 5.0 on Alpha/VMS 6.1)
