Abstract

This document is an overview of scripting for the WASD VMS HyperText Transport Daemon. 

Also see WASD Technical Overview for information on using the WASD VMS Hypertext Services and WASD Hypertext Environment for information on using the WASD VMS Hypertext Services and WASD Nuts and Bolts for technical information on server design and implementation.
