This software is packaged as a self-extracting PCSI kit.  If you are not
running VMS V7.1-2 or later you must have installed one of the two patch
kits listed below in order to install this kit.  These patch kits update
your PCSI software to at least the same functionality available with the
version that ships with VMS V7.1-2 which is needed by this kit.

VMS62TO71U2_PCSI-V0200 for Alpha
VMS62TO71_PCSI-V0200 for VAX
