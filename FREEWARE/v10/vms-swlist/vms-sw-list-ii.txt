The OpenVMS Software List II
Version 8.00
J.C. Higgins





The following programs are available as described below. If you have any 
additions / suggestions / corrections to make then you can contact me at 
Chris@csvax1.ucc.ie. Please submit all Application-specific questions 
to the originators of the applications. Availability codes are at the 
end of the list.

A current version of this file may be retrieved via Anonymous Ftp from 
csvax1.ucc.ie (143.239.1.10). It is available either as a .DVI file or 
as plain ascii text or as postscript. All versions are available as either
compressed or uncompressed versions. The compressed versions are GZIP'ped

The files are available , either in raw format or GZIP'ped , to get the GZIP'ped
version append `-gz' to the filename.  
DVI: [.vms.sw-list]vms-sw-list-ii.dvi   
Postscript: [.vms.sw-list]vms-sw-list-ii.ps  
Ascii Text: [.vms.sw-list]vms-sw-list-ii.txt  

Alternatively , the `.ps' and `.txt' versions are available in one file ZIP'ped.
ZIP: [.vms.sw-list]vms-sw-list-ii.zip 

You can now get at the software list via the World-Wide Web.
The URL is either of :
http://csvax1.ucc.ie/ 
http://csvax1.ucc.ie/www/vms_sw_list/sw_list.html

 .ZOO 		 Zoo archiver		 ZOO		 ZOO -x 
 .HQX 		 Macintosh Format	 BinHex 
 .SHAR 		 Unix format		 UNSHAR 
 .TAR 		 Unix format		 TAR		 TAR xf 
 .TAZ 		 Short hand for .TAR_Z	 UNCOMPRESS and TAR 
 .SIT 		 Macintosh format.	 Stuffit 
 .LZH 		 LHARC Archiver		 LHARC		 LHARC xe
 .LZ , .LHA 	 Amiga version of LHARC	 LHARC		 LHARC xe
 .BOO 		 BOO Archiver		 BOO		 UNBOO 
 .ARJ 		 ARJ Archiver		 ARJ		 UNARJ 
 .UUE 		 Uuencode		 UUENCODE	 UUDECODE 
 .VMS_SHARE , .SHARE , .nn ,
