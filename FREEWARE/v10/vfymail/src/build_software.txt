The following commands can be used to build this software:

    $ BASIC  vfymail_main
    $ BASIC  vfymail_misc
    $ BASIC  vfymail_subs
    $ BASIC  vfymail_tree
    $ LINK  vfymail/OPTION


If you have MMS, then type:

    $ MMS /DESCRIPT=VFYMAIL


To test:

    $ SET COMMAND TEST_CLD
    $ VFYMAIL
