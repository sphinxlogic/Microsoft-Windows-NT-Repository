To try this software:

   1. COPY the appropriate file (VFYMAIL.EXE_VAX or VFYMAIL.EXE_AXP)
      to VFYMAIL.EXE.

   2. Edit the following line in the VFYMAIL.CLD file:

         image       "VFYMAIL"

      to point to the directory containing the VFYMAIL.EXE file.  For example,
      "SYS$DISK:[]VFYMAIL".

   3. Type "SET COMMAND VFYMAIL"

   4. Type "VFYMAIL /STAT"


The file VFYMAIL.HLP contains additional information on this utility.
