MXRN, TOOLS, News Readers version 6.18-27

MXRN is an X-based interface to the USENET news system that uses the
NNTP remote news server for accessing newsgroups and articles.  By using
the NNTP server, users can read news from personal workstations with the
program accessing the news groups and articles from a central repository
on the local area network.
