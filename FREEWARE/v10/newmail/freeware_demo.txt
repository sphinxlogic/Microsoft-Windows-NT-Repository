To try this software:

   1. COPY the appropriate file (NEWMAIL.EXE_VAX or NEWMAIL.EXE_AXP)
      to NEWMAIL.EXE.

   2. Edit the following line in the NEWMAIL.CLD file:

         image       "NEWMAIL"

      to point to the directory containing the NEWMAIL.EXE file.  For example,
      "SYS$DISK:[]NEWMAIL".

   3. Type "SET COMMAND NEWMAIL"

   4. Type "NEWMAIL"

   5. Type "NEWMAIL/FULL"


The file NEWMAIL.HLP contains additional information on this utility.
