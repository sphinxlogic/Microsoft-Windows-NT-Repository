XPOOL 1.3 port under OpenVMS
=============================

MAKE.COM will build Xpool with VAX C 3.2 under OpenVMS/VAX and
DEC C under OpenVMS/AXP

COMPILE.COM and LINK.COM command procedures are also provided.

Define xpool symbol:

$ xpool :== $disk:[directory]xpool

Thanks to Martin P.J. Zinser, Gesellschaft f. Schwerionenforschung GSI, 
for OpenVMS/AXP port and trajectory button

........................................................................
                                                          
You can run XPOOL via FREEWARE_DEMO.COM procedure

Sources are available under [.SRC] directory in XPOOL_VAX_AXP.ZIP file
