XMORPH-SEP95, MOTIF TOYS, Digital image warping (morphing) for X window system

Xmorph is a digital image warping program, also known as a "morphing" program. 
It runs under the X Window System.  If imagefile names are provided on the
command line, then those images are loaded in.  Otherwise, xmorph generates
images to be used in lieu of images provided by the user.  Images may be loaded
from menus within xmorph .

Xmorph has built-in help pages that answer questions about the details of its
use.  Run xmorph for more information.
