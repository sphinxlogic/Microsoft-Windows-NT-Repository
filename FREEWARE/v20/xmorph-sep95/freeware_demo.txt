You can run XMORPH via FREEWARE_DEMO.COM procedure. You can first read the
built-in help pages. To morph, you need Targa images. Targa files (.TGA) can be
produced by ImageMagick or XV 3.10 image viewers under X window.

Sources are available under [.SRC] directory in XMORPH-18SEP95.ZIP file
