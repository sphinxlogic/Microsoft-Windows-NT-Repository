DECthreads HTTP server for VMS

This is the default 'root' page for the DECthreads HTTP server. It is included in the distribution for the following purposes:

  • Provide a working configuration to verify correct operation of the server.

  • Provide an example configuration as a base for organizing your own server files.

Useful information about the server

  • Understanding server error messages.
  • Overview of server internals
  • Format of configuration file
  • How to set up and write server scripts
  • Access control, host-based or user-based

  • CGI scripts from DCL, DCL_ENV
  • Clickable images, Nagel23

  • Special script function, Invalidate cache

  • Server-side HTML pre-processor
  • Dynamic directory browsing
  • Dynamically loaded MST-based scripts: testcgi or omnimap
  • Fork-based script (unix scripting): CGI environment
