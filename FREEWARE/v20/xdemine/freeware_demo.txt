You can run XDEMINE via FREEWARE_DEMO.COM procedure

Sources are available under [.SRC] directory in XDEMINE.ZIP file

MAKE.COM will build Xdemineur with VAX C 3.2 under OpenVMS/VAX and
DEC C under OpenVMS/AXP

COMPILE.COM and LINK.COM command procedures are also provided.

Define xdemineur symbol:

$ xdemine*ur :== $disk:[directory]xdemineur

Thanks to  Bertrand Greff (Rhone Poulenc Rorer) for a modified algorithm
using RAND function under VAX/C (better randomization).
