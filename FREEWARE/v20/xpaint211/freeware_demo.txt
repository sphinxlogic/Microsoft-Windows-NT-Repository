Xpaint version 2.1.1 - OpenVMS VAX installation -
====================

Executable was built on VAX/VMS 5.5-2 with DECW/Motif 1.1

Ressource file: XPAINT.DAT 

Put it in SYS$COMMON:[DECW$DEFAULTS.USER] , or in your SYS$LOGIN directory.

Palette file: .XPAINTRC

Put it on your working directory

Bugs: the file browser doesn't work under VMS. You must type the file name
for loading or writing an image. The Pattern editor crashes. Other functions
seems to work correctly (with sometimes x toolkit errors).

AXP version is a vested version.

You can run XPAINT via FREEWARE_DEMO.COM procedure (you can load
CENA_ATHIS.GIf, a photo of the CENA's small building in Athis-Mons (near Orly
airport) .

Sources are available under [.SRC] directory in XPAINT211.ZIP file
