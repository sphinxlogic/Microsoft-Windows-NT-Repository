XFIL 2.02 port under OpenVMS    Feb 1995
============================

You can run XFIL via FREEWARE_DEMO.COM procedure

Sources are available under [.SRC] directory in XFIL202.ZIP file


Xfil don't compile with VAX C, use DEC C instead.

Bitmap files may be pointed with XFIL_DIR logical name in order to run Xfil.

The mouse is not in effect, use only the keayboard.
