FLIST V2.2-1, TOOLS, A TPU-based file and directory manager
                                       
   FLIST is a file and directory manager for VMS. It is written in DEC
   TPU and C. It lets you view, delete, rename, etc., files and includes
   support for tagging groups of files for deletion, renaming, etc.
   
   FLIST is written in TPU and C. Complete sources are provided.
